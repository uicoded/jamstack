Core Principles

CDN

Jamstack project does not rely on server-side code. Therefore the static content can be distributed and served directly from CDN.

a href="https://cdn.reviews/">CDN Reviews

  • cloudflare
  • AWS Cloudfront

Git

Stores all static content, simplified production workflow.

Git version control

  • GitHub
  • GitLab
  • Bitbucket

Cache Invalidation

The instant build-to-deploy cycle requires to eliminate possibility of stalled resources

Atomic Updates

Large code base requires to upload huge number of files. Atomic updates insures the system goes live only after all resources are uploaded.

Automatic Builds

Jamstack markup is prebuilt, content changes do not go live outside another **whole** build cycle. This process need to be automated.
